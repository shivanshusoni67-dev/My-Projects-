Basic Trigonometric Ratios
sin θ
= Opposite / Hypotenuse
= y / r  |  Unit circle: y
cos θ
= Adjacent / Hypotenuse
= x / r  |  Unit circle: x
tan θ
= Opposite / Adjacent
= sin θ / cos θ  |  = y / x
cosec θ (csc)
= Hypotenuse / Opposite
= 1 / sin θ  |  = r / y
sec θ
= Hypotenuse / Adjacent
= 1 / cos θ  |  = r / x
cot θ
= Adjacent / Opposite
= cos θ / sin θ  |  = x / y

Pythagorean Identities
Primary
sin²θ + cos²θ = 1
Derived — Tangent
1 + tan²θ = sec²θ
Derived — Cotangent
1 + cot²θ = cosec²θ
Reciprocal & Quotient Identities
Reciprocal
sinθ = 1 / cosecθ
cosθ = 1 / secθ
tanθ = 1 / cotθ
Quotient
tanθ = sinθ / cosθ
cotθ = cosθ / sinθ
Even & Odd (Negative Angle) Identities
Even Functions
cos(−θ) = cosθ
sec(−θ) = secθ
Odd Functions
sin(−θ) = −sinθ
tan(−θ) = −tanθ
cot(−θ) = −cotθ
cosec(−θ) = −cosecθ
Co-function Identities (Complementary)
90° − θ
sin(90°−θ) = cosθ
cos(90°−θ) = sinθ
tan(90°−θ) = cotθ
90° − θ (cont.)
cosec(90°−θ) = secθ
sec(90°−θ) = cosecθ
cot(90°−θ) = tanθ
Sum & Difference Formulas
Sine
sin(A+B) = sinA cosB + cosA sinB
sin(A−B) = sinA cosB − cosA sinB
Cosine
cos(A+B) = cosA cosB − sinA sinB
cos(A−B) = cosA cosB + sinA sinB
Tangent
tan(A+B) = (tanA + tanB) / (1 − tanA·tanB)
tan(A−B) = (tanA − tanB) / (1 + tanA·tanB)
Double & Half Angle Formulas
Double — sin
sin(2θ) = 2 sinθ cosθ
Double — cos
cos(2θ) = cos²θ − sin²θ
        = 2cos²θ − 1
        = 1 − 2sin²θ
Double — tan
tan(2θ) = 2tanθ / (1 − tan²θ)
Half — sin
sin(θ/2) = ±√[(1 − cosθ) / 2]
Half — cos
cos(θ/2) = ±√[(1 + cosθ) / 2]
Half — tan
tan(θ/2) = sinθ / (1 + cosθ)
        = (1 − cosθ) / sinθ
Product-to-Sum & Sum-to-Product
Product → Sum
sinA sinB = ½[cos(A−B) − cos(A+B)]
cosA cosB = ½[cos(A−B) + cos(A+B)]
sinA cosB = ½[sin(A+B) + sin(A−B)]
Sum → Product
sinC + sinD = 2 sin(C+D/2) cos(C−D/2)
sinC − sinD = 2 cos(C+D/2) sin(C−D/2)
cosC + cosD = 2 cos(C+D/2) cos(C−D/2)
cosC − cosD = −2 sin(C+D/2) sin(C−D/2)
